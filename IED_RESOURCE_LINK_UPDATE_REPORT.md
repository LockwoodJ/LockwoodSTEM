# IED Resource Link Update Report

Updated the Lockwood STEM website to include student-facing downloadable PDF resources for each IED aerospace unit.

## Added folders

- `resources/ied/unit0/` through `resources/ied/unit7/`

Each unit folder includes:

- `unit#-student-packet.pdf`
- `unit#-rubric.pdf`
- `unit#-quizzes.pdf`

## Updated pages

- `ied-unit-0.html` through `ied-unit-7.html`
- `ied.html`
- `ied-units.html`

## Added stylesheet

- `assets/ied-resources.css`

## Important privacy note

Teacher guides, answer keys, and quizzes with answer keys were **not** placed in the public website folder. This prevents answer keys from being exposed if the site is uploaded to GitHub Pages or embedded in Google Sites.
