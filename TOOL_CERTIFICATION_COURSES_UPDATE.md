# Individual Tool Certification Courses Update

Added individual tool certification courses and tests for:

- Shop Safety
- Hand Tools
- Power Tools
- 3D Printers
- Soldering & Electronics
- Laser Cutter
- CNC Mill & Router

Each course page includes training modules, equipment covered, hands-on sign-off expectations, and a linked certification test. Each test automatically scores responses, requires 80% or higher, and generates a printable certificate of completion.
