# FabLab Certification Tests and Certificates Update

This update restores end-of-level certification tests for Levels 1–4. Each test contains 25 questions. Students must score 80% or higher to generate a certificate of completion. The certificate can be printed or saved as a PDF from the browser and includes a certificate ID, date, level, score, and student name.

Added pages:

- `fablab-certification/level-1-test.html`
- `fablab-certification/level-2-test.html`
- `fablab-certification/level-3-test.html`
- `fablab-certification/level-4-test.html`

Added assets:

- `assets/certification-test.css`
- `assets/certification-test.js`

Important note: These are static client-side checks appropriate for student training and certificate generation. Restricted or high-stakes assessments should remain in Google Classroom, Canvas, or another authenticated platform.
