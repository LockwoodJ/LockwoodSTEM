Preparation Checklist

Use this checklist before tool use, printing, laser cutting, CNC work, or submitting a request.

File and Design Checks

  • Scale: dimensions match the project requirements.
  • Units: millimeters or inches are correct for the tool workflow.
  • Name: file name includes your name, course, and project.
  • Material: material choice is appropriate and available.
  • Safety: design does not create sharp, unsafe, or fragile hazards.

Lab Behavior Checks

  • Ask first before using unfamiliar machines or materials.
  • Do not change settings unless instructed.
  • Do not remove work from a machine unless it is yours or you are told to.
  • Keep the area clean before, during, and after work.
  • Report issues immediately instead of trying to force a tool to work.
