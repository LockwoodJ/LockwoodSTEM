# IED Media Asset Upgrade Report

Added a media-rich visual layer to the Introduction to Engineering Design aerospace curriculum pages.

## Added
- `assets/ied-media.css` for responsive image galleries, video embeds, media cards, and visual reference blocks.
- Custom local SVG hero visuals for Units 0–7.
- Custom local SVG example visuals for design process, sketching, CAD, drawings, structures, mechanisms, and final design review.
- Media & Examples sections on each Unit 0–7 hub page.
- Compact Visual Reference sections on Unit 0–7 subpages.
- Embedded Watch & Learn video cards using NASA and Autodesk YouTube videos.
- Media library overview cards on `ied.html` and `ied-units.html`.

## Notes
- The local SVG visuals are original generated classroom graphics, so they do not depend on external image files.
- Video embeds require internet access and may be controlled by the host platform or school network filtering.
- The media CSS is linked after existing styles so it integrates with the Lockwood STEM theme.
