Submission Checklist

Use these checks before submitting.

File Access

  • Link opens correctly.
  • Anyone with the link can view it.
  • File is named clearly.

Design Readiness

  • Design is complete.
  • Scale and units are correct.
  • Geometry is clean.

Project Info

  • Project name is clear.
  • Material is selected.
  • Notes explain special needs.
