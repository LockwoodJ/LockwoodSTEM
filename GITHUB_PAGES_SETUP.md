# GitHub Pages Setup

Upload the CONTENTS of this folder to the root of your GitHub repository. Do not upload the folder itself as a nested directory.

Required root-level files/folders:
- index.html
- assets/
- fablab-certification/
- .nojekyll

GitHub Pages settings:
- Source: Deploy from a branch
- Branch: main
- Folder: / root

CSS check after publishing:
Open `/assets/lockwood-polish.css` on your published site. If it loads, your theme assets are in the correct place.

Google Sites: use Full page embed for the cleanest full-width display.
