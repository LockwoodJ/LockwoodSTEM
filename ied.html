IED Expectations

Strong engineering work shows the process, not just the final answer.

Before You Build

  • Define the problem clearly.
  • Identify criteria and constraints.
  • Create sketches before modeling.
  • Plan how your design will be tested.

Before You Submit

  • Check file names and file types.
  • Include screenshots, sketches, or photos.
  • Explain what changed during revision.
  • Submit only when files are ready for review.

Ready to fabricate?

Make sure your file is complete, named clearly, and ready for review before submitting.
