# Agenda Large Box Update

Updated `agenda.html` so the Class Agenda area has more room for longer agenda text and bullet lists created from semicolon-separated spreadsheet cells.

## Changes

- Increased the day-view agenda column width.
- Made the Class Agenda card consume more of the available vertical space.
- Enlarged the Agenda row inside the card.
- Prevented agenda text and lesson titles from being clipped with ellipses.
- Increased agenda bullet spacing and line height for readability.
- Added responsive fallbacks so the page still stacks correctly on smaller screens.
