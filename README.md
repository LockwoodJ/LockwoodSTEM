# Lockwood STEM Unified Site

This repository combines the main Lockwood STEM website and the FabLab Certification Hub into one GitHub Pages-ready site.

## Structure

- Root HTML files are the main Lockwood STEM site.
- `/assets/` contains shared Lockwood STEM branding and global CSS.
- `/fablab-certification/` contains the Certification Hub.
- `/fablab-certification/assets/certification-style.css` contains Certification Hub-specific styling.
- `/fablab-certification/assets/certification-script.js` contains Certification Hub-specific JavaScript.

## Navigation

All main site pages use the same global navigation and include a direct Certifications link.
All Certification Hub pages use the same Lockwood STEM global navigation plus a local Certification Hub quick-link bar.

## GitHub Pages

Publish from the repository root using GitHub Pages. Use relative links; do not convert internal links to root-relative paths beginning with `/`.

## Google Sites

Link your Google Site homepage to the GitHub Pages `index.html`, or directly to key pages such as:

- `agenda.html`
- `courses.html`
- `fablab.html`
- `fablab-certification/index.html`
- `fabrication.html`
