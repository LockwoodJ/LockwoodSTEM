Unit Overview

In Unit 4, you will learn how engineers move from a model or sketch to a documentation package that can be used for fabrication, inspection, and revision. A good part drawing does more than show what something looks like. It communicates size, shape, material, fit, process, and quality expectations.

Essential Question

Manufacturing Communication

How do engineers communicate enough information for a part to be manufactured, inspected, and improved?

Main Outcome

Documentation Package

You will create a drawing and documentation package for an aerospace component.

Learning Targets

  • I can measure parts accurately using appropriate tools.
  • I can explain accuracy, precision, tolerance, and fit.
  • I can create a technical drawing with clear views and layout.
  • I can apply dimensions that support manufacturing and inspection.
  • I can add material, process, and manufacturing notes.
  • I can create a simple bill of materials and revision record.
  • I can prepare a part for fabrication review.
  • I can explain whether a design is manufacturing-ready.

What You Will Submit

  • Measurement practice
  • Caliper checkpoint
  • Tolerance and fit practice
  • Technical drawing setup
  • Dimensioned component drawing
  • Manufacturing notes
  • BOM and revision record
  • Fabrication planning checklist
  • Final reflection
