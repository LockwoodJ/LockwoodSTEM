Level 1 / Hand Tools
Core Access

Hand Tool Certification

Hand tools are used constantly in the FabLab. Learn how to measure accurately, cut safely, shape cleanly, and return tools so the next group can work.

Tools You Will Practice

Screwdrivers

Hand tools for installing and removing fasteners.

Files

Hand tools for shaping, smoothing, and deburring edges.

Box Cutter

Cutting tool for cardboard, foam board, and light sheet materials.

What You Need to Learn

  • Choose the correct tool for the task.
  • Secure material before cutting or filing.
  • Keep hands out of the cutting path.
  • Use minimal blade exposure on box cutters.
  • Return tools to labeled storage after use.
Hand tools can still cause injuries. Most problems happen when students rush, skip clamping, use dull blades, or choose the wrong tool.

Tool Selection Mini-Sim

Use a ruler or caliper, then mark with pencil or scribe.

Student Procedure

  1. Choose the tool that matches the task and material.
  2. Inspect the tool for damage before using it.
  3. Secure the material when cutting, filing, or applying force.
  4. Use controlled motion and keep your other hand out of the path.
  5. Clean up scraps, retract blades, and return tools to storage.
