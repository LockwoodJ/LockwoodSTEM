Lockwood STEM • FabLab Training

Learn the tools. Earn access. Build with confidence.

This certification hub teaches you how to work safely in the FabLab, choose the right tool for the job, prepare files correctly, and complete projects with professional habits.

Your Goal

Become trusted with tools

Certification is not just a quiz. You will learn the process, practice safe setup, and demonstrate that you can use each tool responsibly.

How It Works

Learn → Check → Demonstrate

  • Read the tool training page.
  • Use the simulation or practice check.
  • Pass the knowledge check.
  • Complete the hands-on instructor sign-off.
FabLab Mindset

Safety is leadership

Strong makers protect themselves, their classmates, the equipment, and the workspace. If something looks unsafe, stop and ask.

Start with shop safety, emergency procedures, hand tools, PPE, and responsible workspace habits.
